A simple wordpress plugin to hook a picture to your posts and use it on a configurable slider in your Theme. 

# Features:
- Upload Picture to Post
- Add Caption to Post
- Add Slider for 2x3 panels

# Theme-Usage:
```php
if (function_exists('wfs_show')) { wfs_show(); }
```
